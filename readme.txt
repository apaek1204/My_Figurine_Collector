MFC utilizes Beautiful Soup to read the code off of myfigurecollection.net, parse important pieces of information, and copy them into a csv file.

Requires Beautiful Soup and python2.7 to run.

Input: list of the websites with the figures you'd like to parse

Output: The information in the following order: Manufacturer, MFC Name, Series, Character,  Gender, Scale, Height, Price, 18+?, Cast Off?, Type, Tags, JAN
